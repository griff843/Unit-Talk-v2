# Slack Setup

## Purpose

Slack is for execution communication, alerts, approvals, and escalations.

## Channels

- `#ut2-control-tower`
- `#ut2-contracts`
- `#ut2-build`
- `#ut2-migration`

- `#ut2-incidents`

## Rule

Slack is not a source of truth. Decisions reached in Slack must be captured in Notion and linked from Linear when they affect scope or delivery.
